⚠️ Archivo abierto de forma incorrecta

Estás abriendo la página directamente haciendo doble clic desde tus archivos (`file://`). Los navegadores bloquean la conexión con PHP en este modo.

Para resolverlo:

1. Asegúrate de ejecutar usbwebserver.exe.

2. Copia los archivos a la carpeta root de USBWebserver.

3. Entra desde tu navegador escribiendo:
http://localhost:8080/index.html
